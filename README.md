# 出租车轨迹可视化

## 安装和启动

项目使用 [thinkjs 2.2](https://thinkjs.org/zh-cn/doc/2.2/index.html) 版本开放

```sh
git clone https://github.com/fuyi501/traffic-visualization-web.git
cd traffic-visualization-web
npm install
npm start
```
